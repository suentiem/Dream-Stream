Dream Stream

Download .zip Download .tar.gz View on GitHub

What is it?

Dream stream lets you set up sounds and animations for your stream that react to games

Supported Games

  • League of Legends

How to install

  • Download .zip and extract to any folder
  • Run main.exe
  • Click configuration to open config page
  • Click browser to open application

How to set up with OBS

  • Install CLR browser for OBS (here)
  • Add a layer for CLR browser and set the page to: http://127.0.0.1:9002/app/
